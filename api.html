API Документация

liquar.space предоставляет REST API для интеграции с вашими приложениями и сервисами.

Скоро

API находится в разработке и будет доступен в ближайшее время. Следите за обновлениями!

Планируемые возможности

Получение данных профиля
GET /api/v1/profile/{username}
Получить публичную информацию о профиле пользователя.

Обновление профиля
PUT /api/v1/profile
Обновить данные своего профиля (требуется авторизация).

Статистика
GET /api/v1/stats/{username}
Получить статистику посещений и кликов.

Загрузка файлов
POST /api/v1/upload
Загрузить изображения, видео или музыку.

Аутентификация

API будет использовать токены доступа. Вы сможете сгенерировать токен в настройках профиля.

Authorization: Bearer YOUR_API_TOKEN
